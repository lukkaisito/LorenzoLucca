1. Consignas Prácticas
2.1 Definición del Público Objetivo
Este proyecto web está diseñado para un nicho específico que valora la nostalgia digital y la cultura alternativa.

Rango etario: Principalmente jóvenes adultos de entre 18 y 35 años. Incluye tanto a la "Generación Z" interesada en la estética Y2K y retro-computing, como a "Millennials" que buscan recrear su experiencia original en la web de mediados de los 2000.

Nacionalidad: Global, con un énfasis en usuarios de habla hispana e inglesa debido a la naturaleza de las referencias musicales y culturales (Grunge, Rock Alternativo).

Ubicación geográfica: Contexto mayoritariamente urbano. Usuarios con acceso a internet de banda ancha en regiones con una cultura de consumo de medios digitales e interés en subculturas musicales.

Variables adicionales:

Nivel de alfabetización digital: Medio-Alto. Usuarios que se sienten cómodos navegando en interfaces no convencionales y que valoran la personalización (DIY - Do It Yourself).

Dispositivos de acceso: Principalmente computadoras de escritorio y laptops, ya que la estética de SpaceHey/MySpace depende de una visualización amplia para apreciar el diseño de columnas y elementos flotantes.

Ocupación/Intereses: Estudiantes de artes, informática, músicos y entusiastas de la cultura "underground".

2.2 Registro de Inspiraciones y Proceso Creativo
Design Inspirations
SpaceHey / Early MySpace: The primary visual reference. I chose this to evoke a sense of digital nostalgia, utilizing a "raw" HTML/CSS look that prioritizes personal expression over modern, sterilized UI standards.

90s Grunge Culture: The color palette (dark grays, deep reds) and high-contrast, grayscale imagery are inspired by 90s album covers (Nirvana, Alice in Chains). This provides a moody, "anti-design" atmosphere.

Brutalist Web Design: The use of unpolished elements, visible borders, and typewriter fonts (Courier New) serves to create a rugged, authentic feel that contrasts with today's "flat" design trends.

Creative Process Documentation
The creative process began with the goal of breaking the "modern web" mold. Initially, a standard blue MySpace theme was considered, but it was discarded in favor of a Grunge/Dark aesthetic to better align with the user's specific interest in 90s rock.

Key decisions included:

Layout evolution: I moved from a standard single-column layout to a classic two-column profile split, which is the hallmark of 2000s social media.

Constraint-based design: I intentionally used "outdated" elements like the <marquee> tag and CSS blink animations. While technically deprecated in modern professional environments, they are essential here to achieve historical accuracy and stylistic irony.

Visual Cohesion: I implemented a global grayscale filter on profile images to ensure that even if the user uploads colorful photos, the "gloomy" grunge consistency remains intact.

Component Refactoring: The "Top Friends" section was originally a vertical list. Through iteration, I refactored it into a horizontal flexbox grid of small squares to match the iconic "Top 8" visual hierarchy, improving scannability without losing the retro feel.
